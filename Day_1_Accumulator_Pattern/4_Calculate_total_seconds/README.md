## Accumulator Pattern: Beginners

# Calculate Total Seconds From List of Minutes

A fitness coach is designing a workout plan where each exercise takes 2, 5, and 10 minutes, respectively. To schedule the session accurately, the coach needs to know the total duration in seconds. Write a program to convert the list of exercise durations from minutes to seconds and sum them up. 
Guide: Start with a total of 0 seconds, multiply each minute value by 60 and add to the total
